# GPS-Aided Inertial Navigation System V1

PCB Design for GPS-Aided Inertial Navigation System
* MCU: STM32F103C8T6
* IMU: MPU9250 (I2C)
* GPS: NEO-6M (UART)
* Pressure/Temp: BMP180 (I2C)

Communication Protocol: CAN
